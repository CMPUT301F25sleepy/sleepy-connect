# Sleepy Event Management App

Have you ever wanted to host an event, but had nowhere to post it? Look no further than Sleepy event management app. 

With our app you will be able to create, share, join and manage events. Using our lottery system, you can select people to join your event at random!

```
Follow these steps to download and run our app:
1. Clone the repository using git clone
2. Open the project in Android Studio
3. Sync Gradle and click run to get started
```

```
To contribute to our app you will need to do the following:
1. Request to become a contributor
2. Clone the project using git clone
3. Create your own branch
4. Open the branch in Android Studio
5. Make changes
6. Submit a pull request
```
